# 프로젝트 셋업 가이드

## 1. 프로젝트 생성

```bash
npm create vite@latest dalbit-flow -- --template react-ts
cd dalbit-flow
```

## 2. pnpm 전환

```bash
rm -f package-lock.json
pnpm install
```

## 3. 의존성 설치

### 달빛약속
```bash
pnpm add @dalbit-yaksok/core @dalbit-yaksok/exts-standard
```

### 순서도
```bash
pnpm add @xyflow/react
pnpm add dagre @types/dagre
```

### 코드 에디터
```bash
pnpm add @monaco-editor/react monaco-editor
```

### 상태관리 + 라우팅
```bash
pnpm add zustand
```

### 애니메이션
```bash
pnpm add framer-motion
```

### PandaCSS (필수)

## 5. 스모크 테스트

달빛약속 + React Flow가 정상 동작하는지 확인:

```tsx
// src/App.tsx (임시)
import { useEffect, useState } from 'react'
import { YaksokSession, CodeFile, parse } from '@dalbit-yaksok/core'

function App() {
  const [ast, setAst] = useState<string>('')
  const [output, setOutput] = useState<string[]>([])

  const testCode = `나이 = 20\n만약 나이 >= 18이면\n    "성인" 보여주기`

  const runTest = async () => {
    // AST 파싱 테스트
    const codeFile = new CodeFile('main', testCode)
    const { ast: parsedAst } = parse(codeFile)
    setAst(JSON.stringify(parsedAst.toJSON(), null, 2))

    // 실행 테스트
    const logs: string[] = []
    const session = new YaksokSession({
      stdout: (msg) => logs.push(msg),
    })
    session.addModule('main', testCode)
    await session.runModule('main')
    setOutput(logs)
  }

  return (
    <div>
      <button onClick={runTest}>테스트</button>
      <h3>AST:</h3>
      <pre>{ast}</pre>
      <h3>출력:</h3>
      <pre>{output.join('\n')}</pre>
    </div>
  )
}

export default App
```

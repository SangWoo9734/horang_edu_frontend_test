# 에이전트 환경 구성

## 시스템 프롬프트

### CLAUDE.md
프로젝트 전반의 컨텍스트를 제공하는 문서.

포함 내용:
- 프로젝트 개요 (달빛흐름 — 코드 ↔ 순서도 시각화 도구)
- 기술 스택 (Vite 8, React, PandaCSS, React Flow, Monaco, @dalbit-yaksok/core)
- 달빛약속 API 활용 맵
- AST → 순서도 노드 매핑 규칙
- 프로젝트 디렉토리 구조
- 디자인 토큰
- 코드 컨벤션

## 에이전트 활용 원칙

1. **아이디에이션**: 에이전트와 브레인스토밍 → 호랑에듀 실제 서비스와 대조하여 중복 검증 → 평가 기준 분석 기반 최종 결정
2. **설계**: 에이전트에게 초안 위임, 달빛약속 API 문서와 대조 검증
3. **구현**: 컴포넌트 단위로 코드 생성 위임, AST 파싱/React Flow 연동은 직접 테스트 검증
4. **교정**: 기존 서비스 중복, 기획 방향성 오류 시 즉시 피드백하여 수정
